# Language-TSS Package for Visual Studio Code

Provides syntax highlighting for [Level-2/Transphporm](https://github.com/Level-2/Transphporm) (Transformation Style Sheets - Revolutionising PHP templating) language ported from [solleer/language-tss](https://github.com/solleer/language-tss).

## About TSS

**Transphporm** is a powerful templating language that lets you write transformations as CSS-like stylesheets. This extension provides syntax highlighting for `.tss` files in Visual Studio Code.

- 🎨 **Syntax Highlighting** for TSS template files
- 📝 **Language Support** for Transformation Style Sheets
- ⚙️ **Auto-pairing** for brackets and quotes
- 💬 **Comment Support** for both line and block comments

## Installation

1. Open Visual Studio Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "TSS Syntax Highlighting"
4. Click Install

Or install via command line:
```bash
code --install-extension Ivko909.tss
```

## Features

- **Syntax Highlighting** - Proper color coding for TSS language constructs
- **Smart Bracket Matching** - Auto-closing brackets and quote pairs
- **Block Comments** - Full support for `/* */` style comments
- **Line Comments** - Full support for `//` style comments
